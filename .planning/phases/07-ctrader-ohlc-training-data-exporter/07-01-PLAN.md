Create `ctrader-projects/OhlcTrainingExporterV10/src/OHLC Training Exporter v1.0.Lifecycle.cs` as partial class `OhlcTrainingExporterV10`. Follow the Wyckoff Lifecycle file structure exactly, but with OHLC-only payload. Implement:

1. **Socket management** (same as Wyckoff):
   - `ConnectSocket()` — TCP connect to SocketHost:SocketPort, send connection_hello JSON, set _socketState
   - `DisconnectSocket()` — close stream and client, set _socketState
   - `SendSocketLine(string json)` — write UTF8 bytes + newline to _networkStream, increment _droppedEventsTotal on failure
   - `SendHeartbeat()` — send `connection_heartbeat` JSON same format as Wyckoff

2. **Date handling** (exact same as Wyckoff):
   - Parse `StringDate` with `DateTime.TryParseExact(StringDate, "dd/MM/yyyy", CultureInfo.InvariantCulture, DateTimeStyles.None, out customDate)`
   - Switch on `LoadTickFrom_Input` for Today/Yesterday/Before_Yesterday/One_Week/Two_Week/Monthly/Custom
   - Error popup on invalid date

3. **Initialize()**:
   - Call `ConnectSocket()`
   - Create the ParamsPanel (delegate to partial method `CreateParamsPanel()`)
   - Add Export button, Reconnect button, status indicator (use `AddExportButton()` helper)
   - Start a timer for heartbeat at `HeartbeatIntervalMs`

4. **Calculate(int index)**:
   - If `_isManualCsvExportInProgress`, call `ExportCsvData(index)`
   - On `IsLastBar`, call `SendSocketData(index)` if socket connected

5. **ExportCsvData(int index)**:
   - Get the bar's open time: `Bars.OpenTimes[index]`
   - Check if bar is within the selected date range (>= fromDateTime)
   - Build the history export record via `BuildHistoryRecord(index)`
   - Call `AppendDirectHistoryJsonl(record)`

6. **BuildHistoryRecord(int index)** — returns a Dictionary<string, object>:
   ```
   {
     "schema": HistoryFileSchema,
     "source": EventSource,
     "source_instance": SourceInstanceName,
     "event": ExportEventName,
     "event_id": BuildEventId(symbol, barTime),
     "instrument": Symbol.Name,
     "timestamp": barTime.ToString("o"),
     "bar_closed": index < Bars.ClosePrices.Count - 1,
     "bar": {
       "open": Bars.OpenPrices[index],
       "high": Bars.HighPrices[index],
       "low": Bars.LowPrices[index],
       "close": Bars.ClosePrices[index],
       "spread": Symbol.Spread,
       "tick_size": Symbol.TickSize,
       "range_ticks": (int)Math.Round((Bars.HighPrices[index] - Bars.LowPrices[index]) / Symbol.TickSize)
     },
     "source_meta": {
       "symbol": Symbol.Name,
       "timeframe": Chart.TimeFrame.ToString(),
       "history_mode": true
     }
   }
   ```

7. **BuildExportPayload(int index)** — returns live event payload Dictionary:
   ```
   {
     "schema": EventContractSchema,
     "source": EventSource,
     "source_instance": SourceInstanceName,
     "event": ExportEventName,
     "event_id": BuildEventId(symbol, barTime),
     "instrument": Symbol.Name,
     "timestamp": barTime.ToString("o"),
     "payload": {
       "open": Bars.OpenPrices[index],
       "high": Bars.HighPrices[index],
       "low": Bars.LowPrices[index],
       "close": Bars.ClosePrices[index],
       "spread": Symbol.Spread,
       "tick_size": Symbol.TickSize
     },
     "source_meta": {
       "symbol": Symbol.Name,
       "timeframe": Chart.TimeFrame.ToString()
     }
   }
   ```

8. **AppendDirectHistoryJsonl(Dictionary<string, object> record)**:
   - Same pattern as Wyckoff: serialize to JSON, append to file at `Path.Combine(CsvOutputFolder, $"history_ohlc_{SanitizeFileToken(Symbol.Name)}.jsonl")`
   - Use `StreamWriter` with `append: true` and `Utf8NoBom`
   - Use `System.Text.Json.JsonSerializer.Serialize` for JSON output (or manual JSON building like Wyckoff if it uses custom serialization)

9. **SendSocketData(int index)**:
   - If `_networkStream != null` and `IsLastBar`, build live payload and send via `SendSocketLine`

10. **ExportEvent handler** (same pattern as Wyckoff):
    - Disable button
    - Set `_isManualCsvExportInProgress = true`
    - Call `ClearAndRecalculate()`
    - In finally: reset flag, re-enable button

11. **ReconnectEvent handler**: Disconnect then reconnect socket

12. **OnDestroy()**: Clean up socket resources
